# 🎯 Job Sniper
A real-time, agentic job hunting platform built for the Anakin Blitz Hackathon.

## 🚀 Core Features
* **Live Agentic Scraping:** Bypasses stale databases to scrape live postings using the Anakin Wire API.
* **Smart Filtering:** A custom backend keyword algorithm filters out low-relevance positions.
* **Match Score & Freshness Badges:** Surfaces high-fit roles with immediate timestamps.

## 🛠️ Tech Stack
* **Frontend:** Next.js (Sora Font, Tailwind CSS)
* **Backend:** FastAPI, Python
* **Integration:** Anakin Wire API
